 README — Proyecto Final: “Mi Tienda de Vinos”

#Propósito de la página
El propósito principal de esta página es Simular una tienda de vinos, permitiendo:

- Mostrar productos en formato de tarjetas (cards) usando Flexbox.  
- Presentar reseñas de clientes mediante un diseño basado en CSS Grid.  
- Facilitar el contacto de los usuarios mediante un formulario responsivo validado con JavaScript.  
- Aplicar buenas prácticas de HTML semántico, CSS externo, Google Fonts y diseño adaptativo.

#Contenido del proyecto
El proyecto incluye:

-index.html 
  Estructura semántica con las secciones: *header, nav, hero, productos, reseñas, contacto y footer*.

-styles.css 
  Archivo de estilos externo que contiene:
  - Estilos básicos para header, footer y navegación.  
  - Google Fonts correctamente implementadas.  
  - Fondos aplicados a distintas secciones.  
  - Diseño responsivo con Flexbox (productos), Grid (reseñas) y media queries (contacto).

-script.js
  Validación básica del formulario: campos obligatorios y aviso al usuario.

-img/ 
  Carpeta con imágenes de productos (1.jpg, 2.jpg, 3.jpg, 4.jpg).

#Funcionalidades destacadas
- Formulario preparado para integrar con Formspree (endpoint reemplazable).  
- Tarjetas de producto con imágenes proporcionadas automáticamente.  
- Diseño adaptable para móviles, tablets y escritorio.
